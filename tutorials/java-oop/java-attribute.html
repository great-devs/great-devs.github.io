Java OOP | Attribute

Daftar Isi

Atribute itu merujuk ke elemen yang ada pada kelas, sebagai contoh kalo Mobil pasti ada elemen-elemennya semisal ada mesin, warna, merek, dsb. Atribute ini yang akan menyimpan data-data tersebut.


Membuat Atribut

Untuk membuat atribut urutannya. 


[identifier] [data_type] [name]
// Contoh
public String color;
dengan data kelas Car. 

class Car {
    // Ini atribute
    String name;
    String color;
    boolean new;
}
Ket.
  • class | Keyword untuk membuat kelas
  • Car | Nama kelas yang dibuat


Membuat Objek

Untuk membuat objek dari kelas yang telah kita buat, semisal kita ingin membuat objek dari kelas Car diatas. 


Car myCar = new Car();
Jika dengan kelas maka kita bisa membuat objek Car lagi dengan nama yang berbeda, nama harus beda. 

Car momCar = new Car();
Car dadCar = new Car();
Car myCar = new Car();
Untuk cara lain membuat objek. 

new Car();
// Membuat objek
Car car = new Car(new Engine());
Ini biasanya dilakukan di input parameter, jadi butuh objek namun hanya untuk menampung data atau hanya memerlukan suatu objekB untuk membuat objekA.
