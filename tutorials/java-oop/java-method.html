Daftar Isi

Selain atribut ada juga fungsi (Method), fungsi ini yang menjadi utama pada objek. Jika di analogikan dalam kehidupan sehari-hari ibarat mesi penggiling, perlu masukan benda untuk di giling tapi kadang sudah ada benda didalamnya dan tinggal giling saja.


Membuat Fungsi

Untuk membuat fungsi urutannya. 


[identifier] [extra] [data_type] [name] [(optional) input_parameter]
// Contoh
public int totalGrindingTime() {
    ...
}
// Contoh dengan extra
public static void grind() {
    ...
}
dengan data kelas Grinding Machine. 

public class GrindingMachine {
    // Ini atribute
    private Object o;

    // Ini fungsi
    public void enterObjects(Object o) {
        ...
    }
}
Ket.
  • public | Keyword bahwa fungsi ini terbuka
  • void | Fungsi ini tidak mengembalikan informasi/data
  • enterObjects | Nama fungsi
  • o | Objek/benda yang diperlukan untuk di giling
